# CSV Cleaner - Proyecto de Limpieza Basica de CSV en Python

## Descripción

`002-csvCleaner` es un script en Python diseñado para limpiar archivos CSV de forma sencilla.  
Realiza operaciones básicas como:

- Eliminar espacios en blanco al inicio y final de cada campo (trim).  
- Normalizar texto a minúsculas.  
- Eliminar filas duplicadas.

Es ideal para preparar datos antes de análisis o cargas a bases de datos.
